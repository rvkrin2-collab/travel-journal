# Travel Journal

Личный экспедиционный журнал путешествий.

Первая версия: Кыргызстан 2026.

## Редакторское правило: сначала место, потом ритм

При сборке главы нельзя переставлять фотографии только потому, что они визуально хорошо сочетаются.

Перед storyboard необходимо:

1. сопоставить каждый выбранный кадр с реальной локацией и моментом маршрута;
2. проверить `public_id` по `dayNN-photos.json`, `dayNN-analysis.json`, авторским подписям и маршруту;
3. держать сцены в фактическом порядке дня;
4. не смешивать разные места в одном блоке только из-за похожего цвета, формы или настроения;
5. если локация кадра сомнительна — не использовать его как ключевой переход и явно отметить сомнение.

Главное: география дня важнее визуальной рифмы. Сначала факт и маршрут, потом журнальный ритм.
